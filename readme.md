### install
```
git clone git@github.com:zerebom/dotfiles.git
make install
```

### how to install on Linux
```zsh

#1. install zsh
sudo apt-get install zsh

#2. install zplug
curl -sL --proto-redir -all,https https://raw.githubusercontent.com/zplug/installer/master/installer.zsh | zsh

#3 .zsh
```

### delete
```
make clean
```
